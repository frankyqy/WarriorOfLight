# WarriorOfLight
这是一个《博德之门3》的职业模组，设计的原型为《最终幻想14》。

每个职业的特性和技能会尽量保持和FF14中的一致（VFX和SFX除外，真的不会弄）。

当然，数值是用脚填的，所以完全没有平衡性可言。

另外，因为是光之战士，所以只会打架，在探索和交流方面的技能只是够用的水平。
